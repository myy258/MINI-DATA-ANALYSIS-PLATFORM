# Mini ML Platform
## 小型机器学习平台
本平台旨在可以快速查看数据分布和进行机器学习任务，可模型模块化。内置少量机器学习模型，统计分析方法和数据可视化。可导入excel文件。无法在本平台处理数据。

## 使用环境

- **Python版本**: 3.10.19
- **Conda**: Anaconda

## 主要模块版本

- **pandas**: 2.3.3
- **numpy**: 2.2.6
- **matplotlib**: 3.10.9
- **openpyxl**: 3.1.5
- **PyQt5**: 5.15.11
- **scikit-learn**: 1.7.2
